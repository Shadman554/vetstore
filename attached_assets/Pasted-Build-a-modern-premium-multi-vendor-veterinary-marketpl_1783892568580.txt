Build a modern premium multi-vendor veterinary marketplace web application similar to Shopify Marketplace and Amazon Marketplace, specifically designed for veterinary clinics, pet stores, veterinary pharmacies, and pet food distributors.

The platform should allow multiple vendors to create and manage their own stores inside one marketplace.

Main Requirements

• Multi-vendor architecture
• Separate dashboard for every clinic/store
• Marketplace owner dashboard
• Customers can purchase products from multiple vendors
• Commission-based marketplace
• Responsive design
• Fast performance
• SEO optimized

Design

Create a clean, elegant, premium medical-inspired UI using white, blue, and green colors with rounded components, smooth animations, modern typography, and responsive layouts. The experience should feel as polished as Shopify while remaining simple and intuitive.

Architecture

Use a scalable architecture that supports thousands of vendors and hundreds of thousands of products. Build with reusable components, role-based authentication (Admin, Vendor, Customer), secure APIs, image optimization, lazy loading, and a database designed for marketplace operations.

The marketplace owner earns a configurable commission from every successful order while each vendor only has access to their own products, orders, customers, and analytics.
